Three-dimensional test patterns for micro/nanotomography


Test patterns for examining the spatial resolution of 3D x-ray images

Outline

In order to estimate the spatial resolution of 3D images taken with micro/nanotomography, we use test objects on the submicrometer scale prepared by focused-ion-beam milling (FIB). The resolutions along the direction within the tomographic slice plane (in-plane or transverse resolution) and perpendicular to it (through-plane or sagittal resolution) can be determined from their square-wave patterns. Following test patterns are available:

Micrometer to submicrometer patterns

Test pattern 080425AL

Material: aluminum wire, 0.25 mm diameter.
Pattern pitches: 2.0, 1.6, 1.2, 1.0, 0.8, 0.6, 0.5, 0.4, 0.3, 0.2 um.
Resin embedded.


Test pattern 080604D1

Material: diamond, ca. 0.15 mm crystal.
Pattern pitches: 2.0, 1.6, 1.2, 1.0, 0.8, 0.6 um. In-plane only.




Nanometer patterns

Test pattern 130315AL

Material: aluminum rod, ca. 0.12 mm square.
Pattern pitches: 2.0, 1.0, 0.5, 0.40, 0.35, 0.30, 0.25, 0.20 um, 180, 160, 140, 120, 100, 90, 80 nm.


Test pattern 130524AL

Material: aluminum wire, 0.10 mm diameter.
Pattern pitches: 2.0, 1.0, 0.5, 0.40, 0.35, 0.30, 0.25, 0.20 um, 180, 160, 140, 120, 100, 90, 80 nm.
